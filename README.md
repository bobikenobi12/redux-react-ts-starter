# redux-react-ts-starter

A starter project for a React/Redux/TypeScript application.
Chakra UI is used as a component library.
Vite is used as a bundler.

## Getting Started

### Prerequisites

- Node.js
- Yarn

### Installing

```
yarn install
```

### Running

```
yarn dev
```

### Building

```
yarn build
```
